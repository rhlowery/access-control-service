# ACS Frontend (UI)

The ACS Frontend is a React-based Single Page Application (SPA) providing an administrative interface for managing access governance.

## Features
- **Persona Management**: Switching between Requester, Approver, and Admin personas for testing and governance.
- **Access Requests**: Guided workflow for discovering datasets and submitting requests.
- **Audit Viewer**: Real-time visualization of access logs.

## Technology Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Authentication**: OIDC with Backend BFF
